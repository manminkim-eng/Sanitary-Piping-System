💡 TIP — .nojekyll 파일이 안 보이면?
.nojekyll 파일이 업로드 목록에 없을 경우, 저장소에서 직접 생성하면 됩니다:

자주 묻는 질문
  • Q
    저장소 이름을 영문으로 해야 하나요?
    아니요. 한글 이름(위생배관)도 가능합니다. URL에는 인코딩되어 표시되지만 정상 작동합니다. 영문을 원하시면 sanitary-pipe 로 만드세요.
  • Q
    비용이 발생하나요?
    GitHub Pages는 Public 저장소에서 완전 무료입니다. 서버 비용, 도메인 비용 없이 영구 운영 가능합니다.
  • Q
    파일을 수정하면 자동 반영되나요?
    네. GitHub에 파일을 업데이트하면 약 1~2분 후 자동으로 배포됩니다.
  • Q
    커스텀 도메인을 연결할 수 있나요?
    네. Settings → Pages → Custom domain에 본인의 도메인을 입력하면 됩니다.
