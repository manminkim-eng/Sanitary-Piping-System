🏢 건물 용도 + 사용 용도 구분
FU값 결정 → 설계 유량 Q → 모든 관경
• 주거용 vs 공용 선택에 따라 동일 기구 수량이라도 FU가 달라져 관경이 변동됨
• 음식점·호텔은 급탕 FU 비중이 높아 급탕관 관경이 커짐
🏗️ 층수 × 층높이
정수두 계산 → 최상층 잔류압 + 환탕 자동판정 + 수직관 층수보정
• 정수두 = 층수 × 층높이 × 9.81 kPa/m
• 최상층 잔류압 = 급수압 - 정수두 - 마찰손실 → 100kPa 미만 시 경고
• 층수×층높이×1.3 > 15m → 환탕 배관 의무 자동 판정
• 6층 초과 → 배수 수직관 +1단계, 16층 초과 → +2단계 자동 상향
⚡ 최대 허용 유속 (m/s)
유속기준 관경 역산 → FU기준 vs 유속기준 → 큰 값 채택
• D(mm) = √(4Q / π·V) 공식으로 최소 관경 역산
• 유속 낮출수록 → 관경 증가 / 유속 높일수록 → 관경 감소
• 1.5m/s 초과 설정 시 수격 발생 위험 경고 표시
